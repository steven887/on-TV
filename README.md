## *How i can  use it* ??  🤔 🤔
   
* **Make sure you have MPV player installed in your system**

  ```
  which mpv
  ```
  if output nothing, you must install it first
  
  **Ubuntu / Debian**
  ```
  sudo apt update
  sudo apt install mpv
  ```
  
  **Manjaro / Arch linux**
  ```
  sudo pacman -S mpv
  ```
  
* **Clone this repo**

  ``` 
  git clone https://github.com/steven887/on-Tv 
  ```
 
* **cd to on-Tv directory & change permission**

  ```
  cd on-Tv/
  chmod +x on-Tv
  ```

* **Execute the file**

  ```
  ./on-Tv
  ```
